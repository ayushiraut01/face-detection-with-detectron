# 🎯 Face Detection with Detectron2

> **A powerful Computer Vision project** built using Facebook AI's Detectron2 framework to perform accurate real-time human face detection in images, videos, and live webcam streams.

This project leverages deep learning-based object detection models such as **Faster R-CNN** and **Mask R-CNN** to identify faces with high precision, even in complex environments. It demonstrates the practical use of **PyTorch**, **Detectron2**, and **OpenCV** to build scalable and efficient AI-powered applications.

---

## 🚀 Project Overview

Face detection is an essential component in many AI-driven applications such as surveillance systems, facial recognition platforms, security systems, and smart attendance tracking.

This project implements a **deep learning-based face detection pipeline** that detects and highlights human faces in real-time using bounding boxes.

### The system can process:

| Input Type | Description |
|------------|-------------|
| 🖼️ **Images** | Static image face detection with bounding boxes |
| 🎬 **Videos** | Frame-by-frame face detection in video files |
| 📷 **Live Webcam** | Real-time detection from live camera streams |

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ayushiraut01/face-detection-with-detectron.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd face-detection-with-detectron
```

### 3️⃣ Install Dependencies

```bash
# Install PyTorch
pip install torch torchvision

# Install OpenCV
pip install opencv-python

# Install Detectron2
pip install detectron2
```

---

## ▶️ How to Run the Project

```bash
python face_detection_with_detectron.py
```

> ✅ The system will automatically detect faces and display **bounding boxes** around them in real-time.
